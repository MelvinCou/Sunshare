Sunshare
Lecture du fichier d'initialisation

Introduction

Le but de ce test est de vérifier s'il est possible de lire le contenu d'un fichier pour paramétrer la connexion à la base de données et ainsi d'éviter de devoir compiler les sources à chaque nouvelle configuration.

Utilisation

Récupérer les binaires

Bibliothèques de Qt : 

$ sudo apt-get install qtbase5-dev-tools qtbase5-dev

Récupérer les codes sources

$ git clone https://gitlab.com/p2022_FI/4-Sunshare.git

Compilation des sources

Dans le dossier du projet : 

$ qmake ./TestQsetting.pro
$ make

Exécution

$ ./TestQsetting

Exigences

Le programme le contenu du fichier Sunshare.ini.

Remarques

Le fichier Sunshare.ini dans lequel est stocké les paramètres de connection à la base de données doit être placé dans le même répertoire que l'exécutable TestBdD.
