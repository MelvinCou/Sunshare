Sunshare
Application finale

Introduction

Ce programme reprend toutes les fonctionnalités des autres tests :

Collecter les indicateurs et les envoyer dans la base de données

Calculer les moyennes

Supprimer les données trop anciennes

Lire le fichier de configuration

Utilisation

Récupérer les binaires

Bibliothèques de Qt + driver mySQL : 

$ sudo apt-get install qtbase5-dev-tools qtbase5-dev
$ sudo apt-get install libqt5sql5-mysql

Compiler les bibliothèques pigpio : 

$ sudo apt install python-setuptools python3-setuptools
$ wget https://github.com/joan2937/pigpio/archive/master.zip
$ unzip master.zip
$ cd pigpio-master
$ make
$ sudo make install

Récupérer les codes sources

$ git clone https://gitlab.com/p2022_FI/4-Sunshare.git

Compilation des sources

Dans le dossier du projet : 

$ qmake ./ApplicationQt.pro
$ make

Exécution

$ sudo ./ApplicationQt

Exigences

Le programme ne doit pas afficher d'erreur lors de l'envoi des requêtes SQL. Les données envoyées sont visibles sur phpMyAdmin.

Remarques

Les programmes RecupererTIC et LectureCompteur doivent être opérationnels. Le fichier Sunshare.ini dans lequel est stocké les paramètres de connection à la base de données doit être placé dans le même répertoire que l'exécutable TestBdD. Le programme n'a pas de fin, pour le terminer, utiliser le raccourci clavier ctrl+C.
