Slot lançant l'acquisition des valeurs moyennes afin de les stocker dans d'autres tables sur le long terme.

Fonctionnement global des moyennes : Par exemple, pour les moyennes quotidiennes, la moyenne est calculée de la veille à aujourd'hui. La technique utilisée est de décrémenter le jour, sans toucher à l'heure.
